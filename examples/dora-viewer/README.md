# DORA Viewer

A visual dataflow graph viewer built with Makepad for exploring DORA dataflow definitions.

## Quick Start

```bash
cargo run --bin dora-viewer
```

## Features

- Load and visualize YAML dataflow definitions
- Interactive node graph with drag, zoom, and pan
- Filter nodes by category (MaaS, TTS, Bridge)
- Search and batch toggle ports by name
- Ctrl+click to enable/disable nodes and ports
- Customize node shapes and edge styles
